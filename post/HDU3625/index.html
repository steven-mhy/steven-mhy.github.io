HDU3625 Examining the Rooms

2019-07-13

考虑把门iii和门iii中的钥匙指向的门jjj连一条有向边。
打开一扇门，那么这个门处在的环内的所有门都可以被打开，所以图中不可能超过kkk个环。
总共的排列数为n!n!n!，题目所求的是n!n!n!个排列中组成≤k\le k≤k个环的数量，即第一类斯特林数。
考虑到111号门不能炸，用总数Su(n,i)S_u(n,i)Su​(n,i)减去111号点单独成环的方案Su(n−1,i−1)S_u(n-1,i-1)Su​(n−1,i−1)

答案就是：

∑i=1k(Su(n,i)−Su(n−1,i−1))n!\frac{\sum^k_{i=1}(S_u(n,i)-S_u(n-1,i-1))}{n!} n!∑i=1k​(Su​(n,i)−Su​(n−1,i−1))​

代码：

#include <bits/stdc++.h>
#define MAXN 105
using namespace std;
double Su[MAXN][MAXN];
inline void Init(){
    Su[1][0]=0,Su[1][1]=1;
    for (register int i=2;i<MAXN;++i){
        for (register int j=1;j<=i;++j){
            Su[i][j]=Su[i-1][j-1]+(double)(i-1)*Su[i-1][j];
        }
    }
}
int main(){
    Init();
    int t;
    cin>>t;
    while (t--){
        int n,k;
        cin>>n>>k;
        double ans=0;
        for (register int i=1;i<=k;++i){
            ans+=(double)(Su[n][i]-Su[n-1][i-1]);
        }
        double fac=1;
        for (register int i=1;i<=n;++i){
            fac=fac*i;
        }
        printf("%.4lf\n",ans/fac);
    }
}
