2.2.2.这种随机化一定是对的，但是时间可能超时（看你脸白不白）：

如果你发现题目中有如下性质，那么你可以用随机化试一试：

要你从a1a2a3....ana_1 a_2 a_3 .... a_na1​a2​a3​....an​选择一些数，构造一个合法序列b1b2b3....bmb_1b_2b_3....b_mb1​b2​b3​....bm​，这种构造方法有很多，因此是SPJSPJSPJ，模板如下：

（bbb为下标序列）

for (register int t=1;t<=100000;++t){
    random_shuffle(b+1,b+1+n);
    //........
    for (register int i=1;i<=n;++i){
        //.........
        if (Check()){
            for (register int j=1;j<=i;++j){
                printf("%d ",a[b[j]]);
            }
            printf("\n");
            return 0;
        }
    }
}
